What is MindEase?

Four features, one calming space

MindEase is a desktop Qt/C++ application built by Phyo Thiha Oo for BMCC students. It combines campus resources, wellness tools, and a private journal in a single thoughtfully designed application.

BMCC Resources
Direct links to tutoring, financial aid, counseling, immigration support, health services, and more — all curated for BMCC students.
Wellness Toolkit
8 themed folders — from mindfulness and sleep to gratitude journaling and music — with practical tools you can use right now.
Private Journal
Write, save, and search personal journal entries. Export as PDF or plain text. Your reflections stay on your device.
Calm Design
Gentle falling-leaf animations, a warm natural color palette, and a quiet layout designed to reduce — not add to — your stress.
C++ / Qt Architecture

Built with OOP principles

MindEase demonstrates all four core object-oriented programming concepts required for CSC211H Honors.

Encapsulation
JournalEntry, JournalStorage — private data, clean public interfaces. The Screen base class hides m_title behind a const accessor.
Inheritance
Home, Journal, Settings, Recommendations, and Toolkit all inherit from the Screen abstract base class.
Polymorphism
MainWindow holds QVector<Screen*> and calls onActivated() + onThemeChanged() on each — runtime dispatch without knowing the concrete type.
Abstraction
Screen declares screenId() as a pure virtual — making it an ABC. Screen cannot be instantiated directly; every screen must supply its own identity.
